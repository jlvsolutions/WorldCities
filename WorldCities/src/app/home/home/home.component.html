WorldCities

A sample web application to demonstrate how to interact with ASP.NET Core 6, Angular, Material, Entity Framework Core and a SQL Database.

It also demontrates the use of a security token with rotating refresh tokens.

Guests are allowed to view city and country information. Users who are registered by joining are allowed to add and edit.

Data is sourced from the free offerings at simplemaps. This progresive SPA (or PWA) is a sandbox. Not responsible for any form of buzzkillmood_good
