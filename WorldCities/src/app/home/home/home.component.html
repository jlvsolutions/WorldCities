WorldCities

A progressive SPA (PWA) web application to sandbox ASP.NET Core 9, Angular, Material Design, Entity Framework, EF Identity, MSSQL and PostgreSQL Databases. This application can be configured to run locally, on a Windows hosting service, or on Amazon Web Services using S3, EC2, RDS, and CloudWatch.

It also excercises a variety of patterns such as security token with rotating refresh tokens.

Guests are allowed to view city and country information. Users who are registered by joining are allowed to add and edit.

Data is sourced from the free offerings at simplemaps. This progresive SPA (or PWA) is a sandbox. Not responsible for any form of buzzkill.
